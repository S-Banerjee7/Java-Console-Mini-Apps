# Java Console Mini-Apps

A collection of interactive, terminal-based Java applications demonstrating state management, input handling, and control flow. 

## Features
* **Banking System:** Simulates ATM deposit/withdrawal logic with active balance tracking.
* **Trivia Game:** Utilizes multi-dimensional arrays and thread pausing for a simulated game show experience.
* **Dice Roller:** Features randomized number generation and ASCII art terminal rendering.
* **Shopping Cart:** Processes user input strings and calculates floating-point cart totals.
